## 📌 Overview

The **Customer Shopping Behavior Analysis** project focuses on understanding customer purchasing patterns through data analytics and visualization. The workflow includes data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis, interactive dashboard development in Power BI, and presenting insights through a professional report and presentation.

This project demonstrates an end-to-end data analytics workflow and highlights practical skills in Python, SQL, Power BI, and business reporting.

## 💡 Key Insights

- Clothing generated the highest revenue among all product categories.
- Most customers are non-subscribers.
- Young Adult customers contributed the highest sales.
- Accessories and Clothing were the most purchased categories.
- Average customer review rating indicates generally positive customer satisfaction.
- Subscription status influences purchasing behavior.

## ▶️ How to Run

### Clone the repository

```bash
git clone https://github.com/your-username/Customer-Shopping-Behavior-Analysis.git
```

### Navigate to the project

```bash
cd Customer-Shopping-Behavior-Analysis
```

### Install Python dependencies

```bash
pip install pandas numpy matplotlib seaborn jupyter
```

### Open the notebook

```bash
jupyter notebook
```

### Run SQL Queries

- Import the dataset into MySQL.
- Execute the SQL script using MySQL Workbench.

### Open Power BI Dashboard

- Launch the `.pbix` file using **Microsoft Power BI Desktop**.
- Refresh the dataset if required.
